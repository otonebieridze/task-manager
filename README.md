# TaskForge

🚀 **A modern task management app inspired by Notion, built with React + TypeScript + Tailwind CSS.**

TaskForge is a task management tool designed to showcase advanced **frontend development skills** with a focus on usability, performance, and responsive design.

---

## ✨ Features

- 📝 **Task Management** – Add, edit, and delete tasks effortlessly.  
- 🎯 **Kanban Boards** – Organize tasks visually with drag-and-drop.  
- 🏷 **Tag System** – Filter tasks by tags, edit or delete tags dynamically.  
- 🎨 **Customizable UI** – Responsive sidebar and Dark Mode support.  
- 📱 **Responsive Layout** – Optimized for desktop, tablet, and mobile.  

---

## 🛠 Tech Stack

- ⚛️ **React** (with Hooks & Context for state management)  
- 🟦 **TypeScript** (type safety & maintainability)  
- 🎨 **Tailwind CSS** (modern utility-first styling)  
- ⚡ **Vite** (fast development build tool)  
- ☁️ **Deployment:** Vercel

## 📂 Project Setup

Clone repo
```bash
git clone https://github.com/otonebieridze/taskforge.git
```

Navigate into project
```
cd taskforge
```

Install dependencies
```
npm install
```

Run development server
```
npm run dev
```
